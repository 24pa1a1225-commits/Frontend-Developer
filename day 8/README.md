# Student Grade Calculator

## Project Description

This is a simple Student Grade Calculator created using HTML and JavaScript.

The application allows users to:
- Enter the student's name.
- Enter marks for two subjects.
- Calculate the total marks.
- Calculate the average.
- Display the grade.
- Validate empty fields.
- Validate marks between 0 and 100.

---

## Technologies Used

- HTML5
- JavaScript

---

## Features

- Student name input
- Two subject marks input
- Sum calculation
- Average calculation
- Grade calculation
- Input validation
- Alert messages for invalid inputs

---

## Grade Criteria

| Average | Grade |
|---------|-------|
| 90 - 100 | A |
| 80 - 89 | B |
| 70 - 79 | C |
| 60 - 69 | D |
| Below 60 | F |

---

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
